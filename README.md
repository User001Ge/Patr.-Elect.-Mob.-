# Election Simulator — FastAPI + Mobile Frontend

ეს ვერსია აერთიანებს:
- FastAPI backend-ს
- მობილურზე მორგებულ პირველ სამუშაო frontend-ს
- იმავე Excel მონაცემთა წყაროს: `data/preferences.xlsx`

## გაშვება

```bash
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

შემდეგ გახსენი:
- Frontend: `http://127.0.0.1:8000/`
- API docs: `http://127.0.0.1:8000/docs`

## რა შეუძლია ამ ვერსიას

- კითხულობს კანდიდატებს და ამომრჩევლებს Excel-იდან
- გაძლევს 3 კანდიდატის არჩევის საშუალებას
- უშვებს ერთჯერად სიმულაციას და Monte Carlo-ს
- აჩვენებს:
  - საბოლოო გამარჯვებულს
  - პირველ ტურს
  - მეორე ტურს
  - გამარჯვების ალბათობებს
  - დამატებით ცხრილებს
  - პრეფერენციების მატრიცას
